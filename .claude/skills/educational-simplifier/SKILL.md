# Skill: Educational Simplifier

## Purpose
Transform complex robotics concepts into beginner-friendly explanations without losing correctness.

## Capabilities
- Rewrite paragraphs at beginner, intermediate, or kid-friendly levels.
- Use examples and analogies.
- Preserve technical accuracy while improving approachability.

## Rules
1. Use simple language only when requested.
2. Preserve engineering meaning.
3. Provide analogy + definition + example structure.

## Do
- Break complex topics into numbered steps.
- Use everyday examples.

## Don’t
- Oversimplify formulas or remove core meaning.

## Example Workflow
Input: “A PID controller adjusts torque based on proportional, integral, and derivative error components.”  
Output: Beginner-friendly explanation with an analogy, e.g., driving a car.
